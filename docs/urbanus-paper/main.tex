\begin{abstract}
  Preliminary sanitary sewer network planning is costly and technically complex, especially in low-income areas and municipalities with limited technical capacity.
  This paper presents the current development of \method, a computational platform that supports the conception, editing, and processing of sewer network layouts from geospatial data.
  The prototype combines a web interface, an editable graph model, elevation enrichment, spatial persistence, and graph-based processing to generate preliminary network alternatives.
  We describe the theoretical basis, related systems, proposed method, architecture, validation status, and limitations of the current solution.
\end{abstract}
